# SafeDocs Brand Theme Guide

Reference guide for colors, shapes, components, and motion so the UI stays consistent with the SafeDocs logo and brand.

## 1) Color Palette

**Primary**
- SafeDocs Blue `#007BFF` — security, trust, professionalism
- SafeDocs Orange `#FF7A21` — accessibility, energy, innovation

**Secondary / Neutral**
- Deep Navy `#0B1D2E` — strong base, text emphasis
- Soft Gray `#F2F4F7` — background panels
- Medium Gray `#C5CBD3` — icons, borders
- Dark Gray `#4A4F57` — body text
- White `#FFFFFF`

**Accent**
- Checkmark Blue `#1F9CEF` — verification/success
+- Amber `#FFA534` — warning/system alerts
+- Error Red `#E5484D` — error states

**Gradients**
- Blue: `linear-gradient(135deg, #007BFF, #1F9CEF)`
- Orange: `linear-gradient(135deg, #FF7A21, #FF8F45)`
- Blue → Orange: `linear-gradient(135deg, #007BFF, #FF7A21)`

## 2) Shapes & Graphic Language
- Core shape: concentric circular grid (vault-inspired) for section dividers, backgrounds, and transitions.
- Supporting motifs: blue/orange arcs, checkmark badge, rounded geometry.
- Shadows: minimal/soft. Radii: 8–16px; circular buttons; soft gray/blue outlines.

## 3) UI Elements
**Buttons**
- Primary: bg `#007BFF`, text `#FFFFFF`, radius 8–12px; hover darken to `#0066CC`.
- Secondary: 2px border `#007BFF`, text `#007BFF`, bg white.
- Accent: bg `#FF7A21` for key actions (Verify, Upload).

**Navigation Bar**
- Background: white or `#F2F4F7`; active link underline as a blue arc; hover `#007BFF`; optional circular icon badges.

**Cards & Blocks**
- Background: white; soft shadow `rgba(0,0,0,0.06)`; rounded edges; optional arc strip/header accent.

**Icons**
- Rounded line icons, two-color strokes (Blue + Gray) with optional checkmark badges.

## 4) Backgrounds & Decorative Elements
- Vault circle pattern: light gray circular grids at 5–10% opacity, top-right or bottom-left.
- Gradients: use the blue/orange gradients above; consider subtle arc overlays to echo the logo.

## 5) Typography
- Headings: Inter / Poppins / SF Pro, bold.
- Body: Inter / Roboto, regular.
- Sizes: H1 48–64px bold; H2 32px semibold; Body 16–18px regular; line-height 1.5–1.7.

## 6) Layout Themes
- Hero: blue→orange gradient arc, vault-circle graphic, “Trusted Document Security” tagline, primary + accent buttons.
- Features: alternate arcs left/right with opposite text alignment.
- Security timeline/process: circular step icons reflecting logo arcs (Upload → Verify → Secure → Access).

## 7) Interaction & Motion
- Micro-animations: button ripple/lock engage; icons rotate 10–15° on hover; checkmark draw-in.
- Page transitions: circular wipe/soft fade; arcs sliding into position.
